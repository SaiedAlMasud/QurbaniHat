# 🐄 QurbaniHat - Trusted Livestock Booking Platform

## 📋 Project Overview

**QurbaniHat** is a modern web application that connects livestock sellers with customers looking for quality animals for Qurbani (sacrifice). The platform provides a seamless experience for browsing, filtering, and booking animals online, making the Qurbani preparation process easier and more reliable.

## 🎯 Purpose

The purpose of QurbaniHat is to:
- Provide a trusted platform for livestock booking during Qurbani season
- Simplify the process of finding and booking quality animals
- Ensure transparency in pricing and animal specifications
- Offer a convenient online booking system for customers
- Connect verified sellers with potential buyers

## ✨ Key Features

### User Features
- **User Authentication**: Register/Login with email/password or Google OAuth
- **Browse Animals**: View all available animals with detailed specifications
- **Filter & Sort**: Sort animals by price (High to Low / Low to High)
- **Animal Details**: View comprehensive information about each animal
- **Booking System**: Book animals with a simple form (authentication required)
- **User Profile**: View and update profile information (name, image)
- **Responsive Design**: Fully responsive interface for all devices

## 🛠️ Technologies Used

- **Next.js 15** - React framework for production
- **Tailwind CSS** - Utility-first CSS framework
- **DaisyUI** - Tailwind CSS component library
- **React Hook Form** - Form validation and handling
- **Animate.css** - Ready-to-use CSS animations
- **React Icons** - Popular icon library
- **Better Auth** - Authentication library for Next.js
- **MongoDB** - NoSQL database
- **MongoDB Adapter** - Database adapter for Better Auth
